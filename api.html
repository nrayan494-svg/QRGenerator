TextQR API

Turn text into a styled, scannable QR code — generate, theme, and export images with a few HTTP calls. This reference covers every endpoint the generator on this site is built on.

Overview

The TextQR API accepts plain text — a message, link, or contact string — and returns a QR code you can render as a canvas image, style with a color preset, and download as a PNG. All requests and responses use JSON.

Base URL:

Base URL 
https://api.textqr.dev/v1

Authentication

Every request needs an API key passed in the Authorization header. Keys are issued from your account dashboard and never expire unless revoked.

cURL 
# Include your key on every request
curl https://api.textqr.dev/v1/qr \
  -H "Authorization: Bearer YOUR_API_KEY"
Keep keys private. Treat your API key like a password — requests made with it are billed and rate-limited to your account.

Rate limits

Free accounts can make up to 100 requests per hour. Each response includes headers showing your current usage.

HeaderDescription
X-RateLimit-LimitRequests allowed per hour
X-RateLimit-RemainingRequests left in the current window
X-RateLimit-ResetSeconds until the limit resets

Generate a QR code

POST/v1/qr

Converts text into a QR code and returns a reference you can style or download.

ParameterTypeDescription
textstringThe content to encode. Required.
sizeintegerOutput size in pixels. Defaults to 300.
Request 
fetch("https://api.textqr.dev/v1/qr", {
  method: "POST",
  headers: {
    "Authorization": "Bearer YOUR_API_KEY",
    "Content-Type": "application/json"
  },
  body: JSON.stringify({
    text: "https://example.com",
    size: 300
  })
});
Response — 201 Created 
{
  "id": "qr_8f3a2c",
  "text": "https://example.com",
  "size": 300,
  "style": "default",
  "created_at": "2026-07-24T09:12:00Z"
}

Apply a style

POST/v1/qr/{id}/style

Applies one of the built-in color presets to an existing QR code. See Style presets for the full list of names.

Request 
fetch("https://api.textqr.dev/v1/qr/qr_8f3a2c/style", {
  method: "POST",
  headers: {
    "Authorization": "Bearer YOUR_API_KEY",
    "Content-Type": "application/json"
  },
  body: JSON.stringify({ style: "cool-purple" })
});

Download image

GET/v1/qr/{id}/download

Returns the finished QR code as a PNG file, ready to save or embed.

cURL 
curl https://api.textqr.dev/v1/qr/qr_8f3a2c/download \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -o my-qr-code.png

Style presets

Six built-in presets are available for the style field. Each pairs a dot color, corner color, and accent color.

NameDescription
90s-neonTeal dots with a bright pink corner accent
valentine-redDeep red dots with a sky-blue corner accent
khaki-punkWarm khaki dots with an olive corner accent
touch-grassGreen dots with a soft yellow corner accent
cool-purpleMagenta dots with a blue corner accent
sophisticated-blackGrey dots with a solid black corner accent

Error codes

Failed requests return a JSON body with a message field explaining what went wrong.

400The request body is missing or malformed — usually a missing text field. 401The API key is missing, invalid, or has been revoked. 404No QR code exists with the given id. 429Too many requests — you've hit your hourly rate limit. 500Something failed on our end. Retry after a short delay.
